# BudgetBasket – Grocery Price Comparison Web App

BudgetBasket is a full-stack web application designed to help users (especially students) save money on groceries by comparing prices across major Australian supermarkets.

💡 Problem:
Many people in Australia shop without knowing:
- which supermarket is cheapest
- how much they could save
- price differences across suburbs

This leads to unnecessary spending.

✅ Solution:
BudgetBasket allows users to:
- Compare grocery prices across Coles, Woolworths, and Aldi
- Select state + suburb for location-based pricing
- Build a basket and calculate total cost across stores
- Identify the cheapest supermarket instantly
- View savings and estimated monthly spending

🛠️ Tech Stack:
Frontend
- React.js
- CSS (custom styling)

Backend
- Node.js
- Express.js

Database
- MongoDB Atlas

⚙️ Features
- Australia-wide support (State + Suburb filtering)
- Dynamic product + price fetching from backend
- Basket cost calculation (real-time)
- Admin panel to add items and prices
- Full CRUD operations
- Savings + monthly estimate insights

How to Run Locally:
Backend
1) cd budgetbasket-backend
2) npm install
3) node server.js

Seed Data
1) node scripts/seed.js

Frontend
1) cd budgetbasket
2) npm install
3) npm start
